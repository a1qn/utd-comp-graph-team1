# UTD Parking Simulator

A Unity 3D game that simulates the experience of finding parking on the UT Dallas campus. Players drive through a campus layout, search for an open parking spot near their destination, and race against the clock to park efficiently.

Built with **Unity 3D (URP)** by a team of five students.

---

## Unity and Tools

- **Engine**: Unity 3D (URP template)
- **Recommended Version**: Unity 2022.3 LTS  
  All team members should use the same minor version to avoid scene and asset issues.
- **Language**: C#
- **Version Control**: Git + GitHub
- **Project Type**: Desktop first, with potential to adapt to other platforms later.
